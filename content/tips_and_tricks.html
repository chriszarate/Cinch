Tips and Tricks

Template variables

There are six template variables:

  • <!-- TITLE -->
  • <!-- TITLE_NO_TAGS -->
  • <!-- CONTENT -->
  • <!-- CUSTOM_HEAD -->
  • <!-- CREATION_DATE -->
  • <!-- MODIFIED_DATE -->

Title

The first occurence of <h1> in a page will be removed and provided to your page template as the “title” variable. If no <h1> is found, the basename of the page will be provided instead.

The “title_no_tags” variable holds the same string with HTML tags removed. This is especially useful for populating your template’s <title> tag.

Content

The content of a page or output of a PHP script is provided to the page template as the “content” variable.

Custom head

If you have additions to the <head> element for a single page, put them in a sidecar file with a “.head” extension:

getting_started.txt
getting_started.head

The contents of this sidecar file will be provided to the page template as the “custom head” variable.

Created and modified dates

These variables provide the creation and modified dates of the source file.

Caching

Text and HTML pages are cached. PHP output is never cached. When editing existing pages manually, remember to delete the corresponding cache file in the “cache” directory.

Troubleshooting

If you are running PHP in safe_mode you may need to set the uid of page.php to match the uid of your PHP process.
